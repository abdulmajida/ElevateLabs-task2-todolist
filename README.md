📝 To-Do List App:
-------------------

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript. This app allows users to add, mark as complete, and delete tasks — with data saved in the browser using localStorage and neat progress bar and task count.


FEATURES:
---------

✅ Add new tasks easily ✅ Mark tasks as completed ✅ Delete tasks from the list ✅ Automatically saves tasks (using localStorage) ✅ Clean and responsive design✅Neat pregress bar and a task count.

 Tech Stack
 ----------

*HTML5 — structure of the app *CSS3 — styling and layout *JavaScript (Vanilla) — logic and interactivity *localStorage — persistent task storage
